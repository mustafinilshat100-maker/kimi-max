# Cron Setup — чтобы боты не засыпали

## Для Kimi (на сервере Oppen Cloud)

### Настроить cron:
```bash
# Каждые 30 минут проверять репозиторий
*/30 * * * * cd ~/kimi-max && git pull origin main >> ~/cron.log 2>&1
```

### Автоматический запуск задач:
```bash
# Проверять tasks/ каждые 30 мин и писать отчёт
*/30 * * * * cd ~/kimi-max && bash check_tasks.sh >> ~/cron.log 2>&1
```

## Для Max (на моём сервере)

### Мой cron (уже настроен):
- Проверяю репозиторий каждые 15 минут
- Проверяю Crypto Hunter логи
- Пишу отчёт каждые 2 часа

## End of Day Report (для обоих)

Каждый вечер (21:00 MСК) оба бота пишут в `review/daily_report_YYYY-MM-DD.md`:

```markdown
# Daily Report - 2026-03-27

## Что сделано
- Kimi: ...
- Max: ...

## Статус задач
- Task #001: In Progress

## Проблемы
- None

## Планы на завтра
- Task #002: ...
```

## Communication Protocol

| Событие | Действие |
|---------|----------|
| Kimi закончил задачу | Пишет тебе в Telegram |
| Max проверил | Пишет тебе в Telegram |
| Есть вопрос | Пишет тебе → ты пересылаешь |
| Срочное | Оба пишут тебе сразу |

**Главное правило:** Боты НЕ пишут друг другу напрямую. Только через тебя.

## Status Check (утром)

Каждый день в 10:00 MСК я проверяю:
1. Crypto Hunter — работает?
2. Репозиторий — есть новые задачи?
3. Kimi — сделал свою работу?

Пишу тебе краткий отчёт.
